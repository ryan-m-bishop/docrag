═══════════════════════════════════════════════════════════════════
  QUICK START
═══════════════════════════════════════════════════════════════════

1. Initialize (if not already done):
   docrag init

2. Add your documentation:
   docrag add brightsign --source ~/docs/brightsign --description "BrightSign docs"

3. List collections:
   docrag list

4. Test search:
   docrag search "your query"

5. Start MCP server (for Claude Code):
   docrag serve
